HTML

Conceitos Basicos

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar o conteúdo de páginas da web.

Com o HTML, podemos organizar textos, imagens, links, tabelas, formulários e outros elementos, que serão interpretados pelos navegadores para exibir a página.

Estrutura Básica de um Documento HTML

Um documento HTML segue uma estrutura padrão:

  • Declaração do tipo de documento.
  • Elemento raiz que envolve todo o conteúdo.
  • Elemento head que contém metadados e links para recursos externos.
  • Elemento body que contém o conteúdo visível da página.

Principais Elementos HTML

Títulos e Parágrafos

  • HTML possui seis níveis de títulos, do mais importante ao menos importante
  • Parágrafos são definidos com a tag <p>.

Links e Imagens

  • Links permitem a navegação entre páginas.
  • Imagens podem ser adicionadas a partir de arquivos locais ou externos.

Listas

  • Listas ordenadas usam a tag <li>.
  • Listas não ordenadas usam a tag <ul>.
  • Itens de lista são definidos com a tag <li>.

Tabelas

  • São usadas para organizar dados em linhas e colunas.
  • Possuem elementos para linhas, células e cabeçalhos.

Formulários

  • Permitem a interação do usuário com a página.
  • Utilizam campos de entrada, rótulos e botões para enviar informações.

Atributos HTML

Os atributos são usados para fornecer informações adicionais sobre os elementos.

  • id: Identificador único para um elemento.
  • class: Define uma classe que pode ser usada para aplicar estilos ou scripts.
  • src: Especifica o caminho de imagens, vídeos ou áudios.
  • href: Indica o destino de um link.
  • alt: Texto alternativo para imagens.

Hierarquia e Aninhamento

Elementos HTML podem conter outros elementos. É importante respeitar a hierarquia e fechar corretamente as tags para que o documento seja válido.

Comentários em HTML

Comentários são usados para documentar o código e não aparecem no navegador.

Eles são escritos entre os sinais <!-- e -->.

Semântica em HTML

O HTML moderno dá importância à semântica, ou seja, ao uso de elementos que trazem significado ao conteúdo.

  • header: Define um cabeçalho.
  • nav: Representa a área de navegação.
  • main: Representa o conteúdo principal.
  • section: Representa uma seção de conteúdo.
  • article: Representa um artigo ou conteúdo independente.
  • footer: Define o rodapé.

Multimídia em HTML

É possível inserir áudio e vídeo diretamente em páginas web:

  • audio: Adiciona arquivos de som, com controles de reprodução.
  • video: Adiciona vídeos, podendo incluir legendas e múltiplos formatos.

Boas Práticas

  • Manter a estrutura do documento bem organizada e indentada.
  • Utilizar atributos de forma clara e objetiva.
  • Dar preferência ao uso de elementos semânticos.
  • Validar o HTML em ferramentas oficiais para garantir acessibilidade e compatibilidade.
