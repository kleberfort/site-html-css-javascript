CSS

Conceito Básico

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de documentos HTML. Com o CSS, é possível controlar a aparência de elementos em uma página web, como cores, fontes, espaçamentos e layout.

O CSS pode ser aplicado de três maneiras:

  • Inline: (embutido) Estilos são aplicados diretamente no elemento HTML usando o atributo "style".

    • Exemplo: <h1 style="color: blue;">Título</h1>

  • Internal: Estilos são definidos na seção de cabeçalho de um documento HTML usando uma tag <style> no cabeçalho do documento HTML.

    • Exemplo:

                <style>
                h1 {
                    color: blue;
                }
            </style>
  • External: Estilos são definidos em um arquivo CSS separado e vinculados ao documento HTML, usando a tag <link>.

    • Exemplo: <link rel="stylesheet" href="styles.css">

Seletores de CSS

Seletores são usados para "selecionar" os elementos HTML que você deseja estilizar. Alguns seletores comuns incluem:

Podemos dividir os seletores CSS em quatro categorias:

  • Seletores Simples: Selecionam elementos HTML diretamente pelo nome da tag, classe ou ID.

    • Exemplos:

    • Tag: h1 { color: blue; }
    • Classe: .minha-classe { font-size: 16px; }
    • ID: #meu-id { margin: 10px; }
  • Seletores Combinadores: Selecionam elementos com base em sua relação com outros elementos.

    • Exemplos:

    • Descendente: div p { color: red; } (seleciona todos os <p> dentro de <div>)
    • Filho: ul > li { list-style-type: none; } (seleciona <li> que são filhos diretos de <ul>)
    • Irmão adjacente: h1 + p { margin-top: 0; } (seleciona o <p> imediatamente após um <h1>)
    • Irmão geral: h1 ~ p { color: green; } (seleciona todos os <p> que são irmãos de um <h1>)
  • Seletores Atributo: Selecionam elementos com base em atributos específicos.

    • Exemplos:

    • a[href] { text-decoration: none; } (seleciona todos os links com atributo href)
    • input[type="text"] { border: 1px solid #ccc; } (seleciona campos de texto)
  • Seletores Pseudo-classes: Selecionam elementos em um estado específico.

    • Exemplos:

    • a:hover { color: orange; } (seleciona links quando o mouse está sobre eles)
    • input:focus { border-color: blue; } (seleciona campos de entrada quando estão focados)

    Comentários em CSS

    Comentários em CSS são usados para explicar o código e são ignorados pelo navegador. Eles podem ser úteis para documentar estilos ou para desativar temporariamente partes do código.

    Os comentários em CSS são escritos entre /* e */.

            /* Este é um comentário em CSS */
        h1 {
            color: blue; /* Este comentário está dentro de uma regra CSS */
        }

    Cores CSS

    O CSS suporta várias maneiras de definir cores, incluindo nomes de cores, valores hexadecimais, RGB, RGBA, HSL e HSLA.

    • Nomes de Cores: Exemplos incluem "red", "blue", "green", etc.
    • Valores Hexadecimais: Exemplo: #FF0000 (vermelho)
    • RGB: Exemplo: rgb(255, 0, 0) (vermelho)

      • Os valores RGB são compostos por três componentes: vermelho, verde e azul e vão do intervalo de 0 a 255.

    • RGBA: Exemplo: rgba(255, 87, 51, 0.5) (o último valor é a opacidade)
    • HSL: (matiz, saturação, luminosidade) Exemplo: hsl(0, 100%, 50%) (vermelho)

      • matiz é a cor percebida: (0 a 360), saturação (porcentagem) é a intensidade (porcentagem) da cor e luminosidade é o brilho da cor.

    • HSLA: Exemplo: hsla(14, 100%, 60%, 0.5)

      • Fundos CSS

      O CSS permite definir fundos para elementos HTML usando a propriedade background. Você pode definir cores de fundo, imagens de fundo e gradientes, entre outros estilos relacionados ao fundo.

      • Cor de Fundo: Use a propriedade background-color para definir uma cor de fundo.
        Pode se usar: nomes de cores, valores hexadecimais, RGB, RGBA, HSL e HSLA.

        • Exemplo: body { background-color: lightblue; }

      • Imagem de Fundo: Use a propriedade background-image para definir uma imagem como fundo.

        • Exemplo: div { background-image: url('imagem.jpg'); }

      • Outras Propriedades de Fundo:
        • background-repeat: Controla se a imagem de fundo se repete (valores: repeat(padrão), no-repeat, repeat-x, repeat-y).

          • Exemplo: div { background-image: url('imagem.jpg'); background-repeat: no-repeat; }

        • background-size: Define o tamanho da imagem de fundo (valores: auto, cover, contain, ou dimensões específicas).

          • Exemplo: div { background-image: url('imagem.jpg'); background-size: cover; }

        • background-position: Define a posição inicial da imagem de fundo (valores: top, bottom, left, right, center ou coordenadas específicas).

          • Exemplo: div { background-image: url('imagem.jpg'); background-position: center; }

        • background-attachment: Controla se a imagem de fundo rola com o conteúdo ou fica fixa (valores: scroll, fixed).

          • Exemplo: div { background-image: url('imagem.jpg'); background-attachment: fixed; }

        • background-clip: Controla como o fundo é recortado (valores: border-box(padrão), padding-box, content-box).

          • Exemplo: div { background-image: url('imagem.jpg'); background-clip: border-box; }

      • Propriedade abreviada: background.

        • Ordem dos valores: background-color | background-image | background-repeat | background-position | background-size

        Exemplo: div { background: lightblue url('imagem.jpg') no-repeat center/cover fixed; }

      Propriedade de Borda CSS

      A propriedade de borda no CSS é usada para definir a aparência das bordas de um elemento. Você pode controlar a largura, estilo e cor da borda.

      • border-width: Define a largura da borda (valores: pixels, em, rem, etc.) ou usando valores predefinidos: thin, medium, thick.

        • A propriedade pode ter de um a quatro valores, permitindo definir bordas diferentes para cada lado. (superior, direita, inferior, esquerda).

        Exemplo: div { border-width: 2px; }

      • border-style: Define o estilo da borda (valores: solid, dashed, dotted, etc.).

        • A propriedade border é uma propriedade abreviada que pode incluir largura, estilo e cor. border: 2px solid blue;

        Exemplo: div { border-style: solid; }

      • border-color: Define a cor da borda (valores: nomes de cores, hexadecimais, RGB, etc.), Permite controlar a cor das quatro bordas ou controlar individualmente (border-top-color, border-right-color, border-bottom-color, border-left-color).

        • Exemplo: div { border-color: blue; }

      • border-radius: Define o raio da borda (valores: pixels, em, rem, etc.).

        • Exemplo: div { border-radius: 5px; }

      A propriedade curta border pode ser usada para definir todas as propriedades de borda de uma só vez.

      Segue a ordem: largura | estilo | cor

      Exemplo: div { border: 2px solid blue; }

      A propriedade border-style é responsável por definir o estilo da borda, como sólida, pontilhada ou tracejada.

      • solid: Borda sólida.
      • dashed: Borda tracejada.
      • dotted: Borda pontilhada.
      • double: Borda dupla.
      • groove: Borda com efeito de entalhe.
      • ridge: Borda com efeito de relevo.
      • inset: Borda que parece estar embutida.
      • outset: Borda que parece estar saindo.
      • none: Sem borda.
      • hidden: Sem borda (semelhante a none, mas usado em tabelas).

      Podemos usar a propriedade border curta para definir todas as propriedades de borda de uma só vez.

      Segue a ordem: largura | estilo | cor

      Exemplo: div { border: 2px dashed red; }

      Margem CSS

      A margem é a área externa de um elemento, que cria espaço entre ele e os elementos adjacentes. Podemos controlar a margem usando as propriedades margin, margin-top, margin-right, margin-bottom e margin-left.

      • margin: Define a margem de todos os lados de um elemento.
      • margin-top: Define a margem superior.
      • margin-right: Define a margem direita.
      • margin-bottom: Define a margem inferior.
      • margin-left: Define a margem esquerda.

      Exemplo: div { margin: 20px; }

      Preenchimento CSS

      O preenchimento (padding) é a área interna de um elemento, que cria espaço entre o conteúdo do elemento e suas bordas. Podemos controlar o preenchimento usando as propriedades padding, padding-top, padding-right, padding-bottom e padding-left.

      • padding: Define o preenchimento de todos os lados de um elemento.
      • padding-top: Define o preenchimento superior.
      • padding-right: Define o preenchimento direito.
      • padding-bottom: Define o preenchimento inferior.
      • padding-left: Define o preenchimento esquerdo.

      Exemplo: div { padding: 15px; }

      A altura, Largura e Largura máxima do CSS

      Podemos controlar a altura e a largura de um elemento usando as propriedades height e width.

      • height: Define a altura de um elemento.
      • width: Define a largura de um elemento.
      • max-height: Define a altura máxima de um elemento.
      • max-width: Define a largura máxima de um elemento.

      Exemplo: div { height: 200px; width: 300px; }

      Modelo de Caixa CSS

      O modelo de caixa CSS descreve como os elementos HTML são exibidos na página. Cada elemento é representado como uma caixa retangular, que consiste em conteúdo, borda, preenchimento e margem.

      • content: A área onde o conteúdo do elemento é exibido.
      • padding: A área entre o conteúdo e a borda do elemento.
      • border: A borda ao redor do elemento.
      • margin: A área externa que cria espaço entre o elemento e os elementos adjacentes.

      A largura total de um elemento deve ser calculada assim:

      • largura total = largura do conteúdo + preenchimento esquerdo + preenchimento direito + borda esquerda + borda direita + margem esquerda + margem direita

      Exemplo: div { width: 300px; padding: 10px; border: 2px solid black; margin: 20px; }


      A altura total de um elemento deve ser calculada assim:

      • altura total = altura do conteúdo + preenchimento superior + preenchimento inferior + borda superior + borda inferior + margem superior + margem inferior

      Exemplo: div { height: 200px; padding: 10px; border: 2px solid black; margin: 20px; }

      Contorno do CSS

      O contorno é uma linha que envolve um elemento, semelhante à borda, mas não afeta o layout da página. Podemos controlar o contorno usando as propriedades outline, outline-width, outline-style, outline-color e outline-offset.

      • outline: Define o contorno de um elemento.
      • outline-width: Define a largura do contorno (valores: pixels, em, rem, etc.) ou usando valores predefinidos: thin, medium, thick.
      • outline-style: Define o estilo do contorno (valores: solid, dashed, dotted, etc.).
      • outline-color: Define a cor do contorno (valores: nomes de cores, hexadecimais, RGB, etc.).
      • outline-offset: Define a distância entre o contorno e a borda do elemento (valores: pixels, em, rem, etc.).

      Abreviação do Contorno do CSS. Exemplo: div { outline: 2px dashed red; }

      Box-Sizing do CSS.
      O Box-Sizing é uma propriedade que altera a forma como a largura e a altura de um elemento são calculadas. Por padrão, a largura e a altura de um elemento são calculadas apenas a partir da área de conteúdo. No entanto, ao definir o Box-Sizing como "border-box", a largura e a altura incluem o preenchimento e a borda, tornando mais fácil o controle do tamanho dos elementos.

      • box-sizing: Define o modelo de caixa de um elemento.
      • content-box: O valor padrão. A largura e a altura incluem apenas o conteúdo.
      • border-box: A largura e a altura incluem o conteúdo, o preenchimento e a borda.

      Exemplo: div { box-sizing: border-box; }

      Texto CSS

      O CSS oferece várias propriedades para estilizar o texto em uma página web. Algumas das propriedades mais comuns incluem:

      • color: Define a cor do texto.
      • font-family: Define a família da fonte (tipografia) do texto.
      • font-size: Define o tamanho da fonte do texto.
      • font-weight: Define a espessura da fonte (valores: normal, bold, bolder, lighter ou números de 100 a 900).
      • font-style: Define o estilo da fonte (valores: normal, italic, oblique).
      • text-align: Define o alinhamento do texto (valores: left, right, center, justify).
      • text-decoration: Adiciona decorações ao texto (valores: none, underline, overline, line-through).
      • text-transform: Controla a capitalização do texto (valores: none, capitalize, uppercase, lowercase).
      • line-height: Define a altura da linha do texto.
      • letter-spacing: Define o espaçamento entre as letras do texto.
      • word-spacing: Define o espaçamento entre as palavras do texto.
      • text-shadow: Adiciona sombra ao texto.
